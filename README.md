## 🚀 Featured Projects

### 🌿 [PlantCareAI — AI Plant Disease Detection & Assistant](https://github.com/sangam962895/-AI-Plant-Disease-Detection)
> AI-powered web app that detects 40+ plant diseases from leaf images and provides treatment guidance via an intelligent chatbot.

- Built a **Transfer Learning CNN** (TensorFlow/Keras) achieving **95%+ accuracy** across 10 crop types with sub-1-second inference
- Designed full end-to-end inference pipeline: image upload → preprocessing (224×224px) → CNN prediction → LLM treatment generation
- Integrated **Google Gemini Pro** with session-based diagnosis context for personalized agricultural Q&A
- Deployed a responsive **Flask + Bootstrap** web app with drag-and-drop upload, diagnosis history, and chatbot panel on Vercel

`Python` `TensorFlow` `Keras` `Transfer Learning` `Google Gemini Pro` `Flask` `NumPy` `OpenCV` `Bootstrap`

---

### 📄 [SummarAIze — Smart Research Document Assistant](https://github.com/sangam962895/SummarAIze)
> AI assistant that reads long research PDFs/TXTs and enables instant summarization, Q&A, and quiz generation.

- Implemented a **RAG pipeline** using **LangChain + FAISS** for semantic search with sub-second retrieval across 100k+ token corpora
- Built a modular **FastAPI** backend handling document chunking, embedding generation, and context-aware LLM response via Groq API
- Designed **"Challenge Me"** mode — AI generates logic-based questions and evaluates reasoning, not just right/wrong answers
- Streamlit frontend delivering source-cited Q&A, auto-generated quizzes, and instant summaries

`Python` `LangChain` `FastAPI` `FAISS` `Streamlit` `Groq API` `Embeddings` `RAG`

---

### 🎬 [Talk2Tube — YouTube Q&A Chrome Extension](https://github.com/sangam962895)
> Chrome Extension that lets users ask natural language questions about any YouTube video using a RAG pipeline.

- Built a **Chrome Extension** with an interactive side panel for real-time Q&A on YouTube video transcripts
- Implemented **FastAPI + LangChain + ChromaDB** backend for transcript indexing, semantic search, and response generation
- Achieved **90%+ timestamped retrieval accuracy** by optimizing chunk size and embedding overlap parameters
- Embedded 10k+ token transcripts in under 5 seconds; chat history persists across sessions

`Python` `LangChain` `ChromaDB` `FastAPI` `JavaScript` `Chrome Extension APIs` `RAG`
